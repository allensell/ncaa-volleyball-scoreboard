# NCAA Volleyball Scoreboard

The purpose of this simple script is to pull the NCAA division I Mens and Womans Volleyball
Games for the day and to present them on a Web page.  It presents the following information

  - The time of the game
  - The stream and/or broadcast network that the game can be seen on
  - If the game is complete the score

If you are running the application it will update every 60 seconds
